Div Tag

The div tag is a block level HTML element. It is used to divide or section of other HTML tags in to meaningful groups.
Example on how to use the div tag.

<div id="navigation">
<ul>
<li><a href="index.html" title="Go Home">Home</a></li>
<li><a href="about.html" title="Learn more about us">About</a></li>
<li><a href="contact.html" title="Contact us">Contact</a></li>
</ul>
</div>

Span Tag

The span tag is an inline HTML element that is used to group a set of inline elements.
Example on how to use span tag.

<h1><span class="classname">Design for the Web Programming</span></h1>

Id attribue

The id attribute is used to label sections or parts of your HTML document.
Example on how to use Id attribute

<div id="navigation">
<ul>
<li><a href="index.html" title="Go Home">Home</a></li>
<li><a href="about.html" title="Learn more about us">About</a></li>
<li><a href="contact.html" title="Contact us">Contact</a></li>
</ul>
</div>

Class attribute

If a page had multiple sidebar chunks, we might want to make one class so that we could use CSS to style each chunk with one rule.
Example on how to use class attribute.

<div class="aside">
<h3>Sidebar Title</h3>
<p>example of class attribute</p>
</div>

Style attributes

Below are some style attributes which you can use to style the code.
  • background-color : This helps in setting the background color for that part of the tag of for the entire body.
  • color : The CSS color property defines the text color to be used.
  • font-family : The CSS font-family property defines the font to be used.
  • font-size : The CSS font-size property defines the text size to be used.
  • border : The CSS border property defines a border around an HTML element.
